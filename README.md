# 実行方法と結果
プロジェクトを実行し、
(読み込むレシピのデータ) (出力したいレシピID) 
の順に入力を与える。コンソールに出力がされる。

入力例：recipe_title.txt 3
出力結果： 3: 杏仁豆腐

# レシピタイトルの保存
recipe_title.txtに保存。

# recipe_title.txt のフォーマット
(レシピタイトル) (レシピ情報のURL)

% cat recipe-data.txt
